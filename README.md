# krakenGame

> 说明：这是早期学习 Unity 时的游戏练习项目，仅用作留档记录。

`krakenGame` 是一个 Unity 游戏工程练习项目。从构建场景看，项目包含开始菜单、主地图、二级地图、三级地图、战斗、对话、进度读取和游戏结束等内容，整体偏向带地图探索和战斗流程的游戏原型。

项目创建提交日期：2018-07-22。这个仓库主要保留了当时 Unity 工程、素材和场景组织方式，适合做个人学习记录。

## 内容概览

- Unity 多场景工程，包含菜单、地图、战斗、对话和结束流程。
- 通过主地图和多级地图组织探索路径。
- 包含脚本、场景、预制体、图片、动画等 Unity 资源。
- 项目结构体现了早期对游戏流程拆分和场景管理的尝试。

## 运行环境

- Unity 2017.2.0f3。
- C#。
- Windows 环境更接近原始开发环境。

## 打开与运行

使用 Unity 2017.2.0f3 或相近版本打开仓库根目录，等待资源导入完成后，可以从以下场景开始查看：

```text
Assets/Scenes/startMenu.unity
```

如果使用较新的 Unity 版本打开，可能会触发资源升级，建议先备份或在单独分支中尝试。

## 开发小结

- 这个项目练习了 Unity 场景组织、地图流程和战斗/对话模块的衔接。
- 多张地图和多个状态场景让项目比单场景 Demo 更接近完整游戏原型。
- 现在看代码和资源可能会有些粗糙，但它记录了当时从“能跑”到“有流程”的探索。
